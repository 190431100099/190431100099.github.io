Informasi Proyek

  • Kategori: Machine Learning

3-Phase Voltage Monitoring with ANN-based Unbalance Detection

Proyek ini merupakan sistem monitoring tegangan 3 fasa berbasis mikrokontroler yang dilengkapi dengan metode Artificial Neural Network (ANN) untuk mendeteksi kondisi ketidakseimbangan tegangan secara otomatis. Sistem ini dirancang untuk meningkatkan keandalan sistem kelistrikan industri dengan mendeteksi gangguan lebih dini.

Data tegangan dari masing-masing fasa (R, S, T) diambil menggunakan sensor tegangan dan diproses secara real-time. Selanjutnya, data tersebut digunakan sebagai input model ANN yang telah dilatih sebelumnya untuk mengklasifikasikan kondisi sistem menjadi normal atau unbalance.

Fitur Utama

Monitoring 3 Fasa Real-Time

Membaca dan menampilkan tegangan pada tiga fasa (R, S, T) secara simultan dan real-time.

Deteksi Unbalance dengan ANN

Menggunakan model Artificial Neural Network untuk mendeteksi ketidakseimbangan tegangan dengan akurasi tinggi.

Analisis Data Tegangan

Data tegangan dianalisis untuk menentukan pola dan kondisi abnormal pada sistem distribusi listrik.

Integrasi Embedded System

Sistem terintegrasi antara sensor, mikrokontroler, dan model machine learning untuk implementasi langsung di lapangan.
